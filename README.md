# Login & Signup Pages

ICSI 418 Homework # 1:
A simple login and signup form built with HTML and CSS.

## Files

- `login.html` - Login page with User ID and Password fields
- `signup.html` - Signup page with First Name, Last Name, User ID, and Password fields
- `styles.css` - Shared stylesheet for both pages

## How to Run

Open `login.html` or `signup.html` in a web browser.

## Features

- Both pages share a single stylesheet (`styles.css`) for consistent styling.
- Forms are centered on the page using `margin: auto`.
- Clean spacing and alignment across all form elements.
- No external frameworks used — plain HTML and CSS only.
- Each page links to the other for easy navigation between Login and Signup.

## Style Guide

This project follows the [Google HTML/CSS Style Guide](https://google.github.io/styleguide/htmlcssguide.html).
